# 项目环境  

## 前端

- 采用vue3框架，搭配vite构建  
- 原生组件搭配ElmentUI
- 后期预计使用TS重构
- ...

## 后端

- 采用springboot框架，mysql作数据库，mybatis协助开发，redis作缓存
- 用户权限认证通过JWT+Spring Security实现
- ...



## 服务端环境

服务端部署在阿里云服务器，使用的Ubuntu系统

# 数据来源

动漫数据来自Bangumi，使用Bangumi公开的API进行获取，

获取后进行本地格式化以供本项目业务使用
